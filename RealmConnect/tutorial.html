What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure and content of a webpage using elements called tags.

Basic HTML Structure

Every HTML document starts with this basic structure:

<!DOCTYPE html> <!-- Tells browser this is HTML5 --> <html lang="en"> <!-- Root element, 'en' means English --> <head> <!-- Contains metadata (not visible) --> <meta charset="UTF-8"> <!-- Character encoding --> <title>My Page</title> <!-- Tab title --> </head> <body> <!-- Visible content goes here --> <h1>Hello World!</h1> </body> </html>

HTML Tags Explained

Tags come in pairs: an opening tag <tag> and a closing tag </tag>

Heading Tags (h1 - h6)

<h1>Largest Heading</h1> <h2>Second Largest</h2> <h3>Third Largest</h3> <h4>Fourth</h4> <h5>Fifth</h5> <h6>Smallest Heading</h6>

Use <h1> for main title, <h2> for sections, <h3> for subsections.

Paragraph Tag

<p>This is a paragraph of text.</p> <p>This is another paragraph.</p>

Links (Anchor Tag)

<a href="https://google.com">Click here</a> <!-- href = hypertext reference (the URL) --> <a href="home.html">Go to Home</a> <a href="#section">Jump to section</a>

Images

<img src="image.jpg" alt="Description"> <!-- src = source (file path) --> <!-- alt = alternative text (for screen readers) --> <img src="icon.jpg" alt="User Icon" width="50" height="50">

Divs and Spans

<!-- div = block container (takes full width) --> <div> <h2>Section Title</h2> <p>Section content</p> </div> <!-- span = inline container (only takes needed space) --> <p>This is <span>highlighted text</span> in a sentence</p>

Lists

<!-- Unordered List (bullets) --> <ul> <li>Item 1</li> <li>Item 2</li> <li>Item 3</li> </ul> <!-- Ordered List (numbers) --> <ol> <li>First step</li> <li>Second step</li> <li>Third step</li> </ol>

Buttons and Inputs

<button>Click Me</button> <input type="text" placeholder="Enter text here"> <input type="password" placeholder="Password"> <input type="email" placeholder="email@example.com">

HTML Attributes

Attributes provide additional information about elements:

  • id - Unique identifier: <div id="header">
  • class - Reusable style name: <div class="container">
  • style - Inline CSS: <p style="color: red;">
  • href - Link destination: <a href="page.html">
  • src - Image/file source: <img src="photo.jpg">
üí° Key Concept: HTML provides the structure and content, while CSS provides the styling and layout.

Common HTML Mistakes

  • Forgetting closing tags: <div>Content ‚ùå ‚Üí <div>Content</div> ‚úÖ
  • Nesting tags incorrectly: <b><i>Text</b></i> ‚ùå ‚Üí <b><i>Text</i></b> ‚úÖ
  • Missing quotes in attributes: src=image.jpg ‚ùå ‚Üí src="image.jpg" ‚úÖ

üé® CSS Fundamentals

What is CSS?

CSS (Cascading Style Sheets) controls how HTML elements look on a webpage. It handles colors, fonts, spacing, layout, and animations.

Three Ways to Add CSS

1. Inline CSS (directly on element)

<p style="color: blue; font-size: 20px;">Blue text</p>

‚ùå Not recommended for large projects

2. Internal CSS (in <head> tag)

<head> <style> p { color: blue; font-size: 20px; } </style> </head>

‚úÖ Good for small single-page sites

3. External CSS (separate file)

<!-- In HTML file --> <head> <link rel="stylesheet" href="style.css"> </head> <!-- In style.css file --> p { color: blue; font-size: 20px; }

‚úÖ‚úÖ Best practice - used in Realm Connect!

CSS Syntax

selector { property: value; another-property: value; } /* Example */ h1 { color: red; /* Text color */ font-size: 32px; /* Text size */ text-align: center; /* Center the text */ }

CSS Selectors

Element Selector

p { color: blue; } /* Styles ALL <p> tags */

Class Selector (with dot .)

.button { background-color: green; color: white; } /* Styles all elements with class="button" */ <button class="button">Click</button>

ID Selector (with hash #)

#header { background-color: black; } /* Styles the element with id="header" */ <div id="header">Header</div>

Multiple Selectors

h1, h2, h3 { color: purple; } /* Styles h1, h2, AND h3 */ div p { color: green; } /* Styles p tags INSIDE divs */

Common CSS Properties

Text Styling

.text-style { color: #ffffff; /* Text color (hex code) */ font-size: 16px; /* Text size */ font-family: Arial, sans-serif; /* Font type */ font-weight: bold; /* bold, normal, 100-900 */ text-align: center; /* left, center, right */ text-decoration: underline; /* underline, none */ line-height: 1.6; /* Space between lines */ }

Colors

.color-examples { color: red; /* Color name */ color: #ff0000; /* Hex code */ color: rgb(255, 0, 0); /* RGB values */ color: rgba(255, 0, 0, 0.5); /* RGB + transparency */ }

Background

.background-style { background-color: #202225; background-image: url('image.jpg'); background-size: cover; /* cover, contain */ background-position: center; }

Box Model (Spacing)

.box { width: 300px; /* Element width */ height: 200px; /* Element height */ padding: 20px; /* Space inside element */ margin: 10px; /* Space outside element */ border: 2px solid black; /* Border around element */ border-radius: 8px; /* Rounded corners */ } /* Individual sides */ .specific-spacing { padding-top: 10px; padding-right: 20px; padding-bottom: 10px; padding-left: 20px; /* Shorthand: top right bottom left */ padding: 10px 20px 10px 20px; /* Shorthand: vertical horizontal */ padding: 10px 20px; }

Display & Layout

.layout { display: block; /* Takes full width */ display: inline; /* Takes only needed space */ display: flex; /* Flexbox layout */ display: none; /* Hides element */ }

Flexbox (The Layout Superpower)

Flexbox makes it easy to align and distribute elements:

.container { display: flex; /* Direction */ flex-direction: row; /* horizontal (default) */ flex-direction: column; /* vertical */ /* Main axis alignment */ justify-content: center; /* center */ justify-content: space-between; /* space between items */ justify-content: flex-start; /* start */ /* Cross axis alignment */ align-items: center; /* center vertically */ align-items: flex-start; /* align to top */ gap: 10px; /* Space between items */ } .item { flex-grow: 1; /* Grow to fill space */ flex-shrink: 0; /* Don't shrink */ }

Hover Effects & Transitions

.button { background-color: blue; transition: all 0.3s ease; /* Smooth animation */ } .button:hover { background-color: darkblue; /* Changes on hover */ transform: scale(1.1); /* Grows 10% */ } /* Other pseudo-classes */ .link:active { color: red; /* While clicking */ } .input:focus { border-color: blue; /* When selected */ }

How Realm Connect Uses CSS

/* Reset default styles */ * { margin: 0; padding: 0; box-sizing: border-box; } /* Discord dark theme */ body { background-color: #202225; color: #dcddde; font-family: 'Whitney', Arial, sans-serif; } /* Flexbox for layout */ .discord-container { display: flex; height: 100vh; } /* Hover effects */ .server-icon { transition: all 0.2s ease; } .server-icon:hover { border-radius: 16px; background-color: #7289da; }
üí° CSS Tips:
  • Start with a reset to remove browser defaults
  • Use classes more than IDs for reusability
  • Use flexbox for modern layouts
  • Add transitions for smooth animations
  • Use developer tools (F12) to test CSS live!
